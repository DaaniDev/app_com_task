# app_com_task

In this repository i have done below tasks:

- Fetch list of movie categories from server
- Fetch list of movies by selected category from server
- Upload new movie

Tech stack used:

- Hilt Dependency Injection
- Kotlin
- Kotlin Coroutines
- Kolin Flows
- Retrofit
- Data Binding
- Binding Adapter
- MVVM
- Repository Pattern
- Sealed Classes
